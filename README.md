DeSmuME(3DS)
=======

Port of DeSmuME DS Emulator for the 3DS.

It's currently a fully interpreted DS emulator, which means the performance is not so great.

How do I use this?
------------------------
Put your ROMs in "root:/DeSmuME/roms"
Assign your ROMs to the buttons shown:

Default (no buttons held) - default.nds

Normal buttons:
L - L.nds
R - R.nds
ZL (New3DS only) - ZL.nds
ZR (New3DS only) - ZR.nds
A - A.nds
B - B.nds
X - X.nds
Y - Y.nds
Start - Start.nds
Select - Select.nds

D-Pad:
Up - d_up.nds
Down - d_down.nds
Left - d_left.nds
Right - d_right.nds

Circle Pad:
Up - cp_up.nds
Down - cp_down.nds
Left - cp_left.nds
Right - cp_right.nds

C-Stick (New3DS only):
Up - cs_up.nds
Down - cs_down.nds
Left - cs_left.nds
Right - cs_right.nds


Using DLDI homebrew:
Create an IMG file in WinImage, and save it to "root:/DeSmuMe/SD.IMG".
